# OpenCV_Beroepsproduct
OpenCV Beroepsproduct WOR_ROBOTS

## Building on Linux

### Build dependencies

Make sure these are installed first:

    cmake - sudo apt-get install cmake
    opencv 3.2.0 and above
    cppcheck 1.86 and above


### Compile

From root:

```
mkdir build
cd build
cmake ../src
make
```
