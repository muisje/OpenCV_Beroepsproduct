# OpenCV_Beroepsproduct
OpenCV Beroepsproduct WOR_ROBOTS

## Building on Linux

### Build dependencies

Make sure these are installed first:

    cmake 2.8 and above
    opencv 3.2.0 and above
    cppcheck 1.86 and above
    boost
    c++ 14

### Compile

From root:

```
mkdir build
cd build
cmake ../
make
```

### Run

from build

```
.\Beroepsproduct
```
